◆ recoverFreeOidList()

void dbDatabase::recoverFreeOidList ( )

Try to recover list of free OIDs in case of its corruption by application. Corruption of free OIDs list can be caused by bugs in application (such as "walking pointer" or out-of-bounds access). It most cases databases should be either recovered by automatic FastDB recovery mechanism (which is involved when database was not closed normally) or by recovery from the backup or XML dump. But if backup is not available and the only thing which is corrupted seems to be OID list, then you can try to use this method.
