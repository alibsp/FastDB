Detailed Description

template<class T>
class dbTimeSeriesProcessor< T >

Time series processor.
Element of time series can be arbitrary type with declared TYPE_DESCRIPTOR and defined time_t time() method

TimeSeries processor constructor

Parameters
databasereference to the database
minElementsInBlockpreallocated number of the elements in the block: array with specified number of elements will be allocated for new block
maxElementsInBlockmaximal number of the elements in the block: block will be splitten if it has maxElementsInBlock elements and new is added to the block
maxBlockTimeIntervalmaximal interval between first and last element in the block, new block will be created if adding new element to the block cause violation of this assumption. If maxBlockTimeInterval is 0, then it is assigned to doubled number of seconds in day multipied on maxElementsInBlock
